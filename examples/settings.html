Custom Palette

_palette.scss

Degree variables are used in creating colors with the light, lighter and dark, darker variations.

// Gray Scale
$black:        #000 !default;
$gray-darker:  #303030 !default;
$gray-dark:    #606060 !default;
$gray:         #909090 !default;
$gray-light:   #aaa !default;
$gray-lighter: #efefef !default;
$white:        #fff !default;

The color palette is built using three primary colors and three secondary. You can customize those six colors to create a new palette.

// Primary
$red:    #de5151 !default;
$yellow: #d1c329 !default;
$blue:   #2ab0ea !default;

// Secondary
$orange: #e69242 !default;
$green:  #8ecb25 !default;
$violet: #bc6bd3 !default;

Degree variables are used in creating colors with the light, lighter and dark, darker variations.

$light-deg:   5% !default;
$lighter-deg: 10% !default;
$dark-deg:    5% !default;
$darker-deg:  10% !default;
-lighter -light -dark -darker
