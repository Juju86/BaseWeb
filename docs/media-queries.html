Media Queries

BaseWeb is built mobile first. What this means in terms of media queries is that each is additive starting from tablet. All styles not wrapped in a media query are applied to mobile directly. The tablet breakpoint is added on top of the base mobile styles, then desktop on top of mobile and tablet, etc.

There are also two supplementary media queries. @mixin media-mobile lets you apply styles to only mobile and @mixin media-small to only vertical oriented mobile.

Unlike most mixins, media mixins don't take parameters. Instead, you pass in the style blocks through brackets like you would with a normal media query.

SCSS 
@include media-small {
  // Vertical Oriented Mobile Only
}

@include media-mobile {
  // Mobile Only
}

@include media-tablet {
  // Tablet and below
}

@include media-desktop {
  // Desktop and below
}

@include media-large {
  // Large desktop and below
}

If you prefer to run your queries differently (desktop first, for example), or if you'd like to add new queries, make sure to review the customization documentation. If possible, try and avoid modifying core BaseWeb files directly as that makes updating to newer versions more difficult.

Retina Media Query

The media query mixin can be used for setting styles specifically to retina screens. This is most commonly used when setting higher resolution background images.

SCSS 
.logo {
  background-image: src('logo.png');
}

@include media-retina {
  .logo {
    background-image: src('logo-2x.png');
    background-size: 100px 50px;
  }
}
CSS 
.logo {
  background-image: src('logo.png');
}

@media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) {
  .logo {
    background-image: src('logo-2x.png');
    background-size: 100px 50px;
  }
}

Variables

$bp-small

The breakpoint to be used for vertical oriented mobile devices. The media feature is expressed using: max-width

$bp-small
  type: unit (pixel)
  default: 480px
Reference: @mixin media-small

$bp-mobile

The breakpoint to be used for all mobile devices and orientations. The media feature is expressed using: max-width

$bp-mobile
  type: unit (pixel)
  default: 760px
Reference: @mixin media-mobile

$bp-tablet

The breakpoint to be used for tablet devices. The media feature is expressed using: min-width

$bp-tablet
  type: unit (pixel)
  default: 760px
Reference: @mixin media-tablet

$bp-desktop

The breakpoint to be used for computer screens. The media feature is expressed using: min-width

$bp-desktop
  type: unit (pixel)
  default: 990px
Reference: @mixin media-desktop

$bp-large

The breakpoint to be used for large computer screens. The media feature is expressed using: min-width

$bp-large
  type: unit (pixel)
  default: 1380px
Reference: @mixin media-large

Mixins

media-small

For applying styles specifically to vertically oriented mobile devices. The @content directive is where your style block will be placed.

@mixin media-small {
  @media screen and ( max-width: $bp-small ) {
    @content;
  }
}

Example Usage

SCSS 
@include media-small {
  // Your style block goes here
}
Reference: $bp-small
