Elements

Elements in BaseWeb are defined as individual HTML components. Things like headers, paragraphs, lists, tables and any other HTML elements that make up the whole of your document. Although most elements in a document are used in groups to form blocks, each element has a base look that functions independently and can be enhanced in either blocks or using modifiers.

Editorial block and elements are used for semantically and visually representing changes in a document. List Mixins

remove-list-styles

Removes the default list styles. This is commonly used when styling a list in the context of a navigation, menu or breadcrumb.

@mixin remove-list-styles

Example Usage

SCSS 
nav ul {
  @include remove-list-styles();
}
CSS 
nav ul {
  list-style: none;
  margin: 0;
  padding: 0;
}
nav ul li {
  margin: 0;
}

make-list-modifier-base

Creates the base styles for a list modifier.

@mixin make-list-modifier-base

Example Usage

SCSS 
%list-base {
  @include make-list-modifier-base();
}

.list-wrap {
  @extend %list-base;
  ...
}
.list-rowed {
  @extend %list-base;
  ...
}
.list-bordered {
  @extend %list-base;
  ...
}
CSS 
.list-wrap,
.list-rowed,
.list-bordered {
  margin: 10px 0;
}
.list-wrap > li:after,
.list-rowed > li:after,
.list-bordered > li:after {
  content: "";
  display: table;
  clear: both;
}
.list-no-clear.list-wrap > li:after,
.list-no-clear.list-rowed > li:after,
.list-no-clear.list-bordered > li:after {
  content: none;
  display: none;
  clear: none;
}
