Functions

This is where we define custom functions for our framework. These functions are global and don't apply specifically to a single element/block.

strip_units

Strips the unit from a value. For example, if you pass it 12px, it will strip off the px unit and return the number 12. This function is used in @function px_to_em and @function em_to_px to return the proper value.

@function strip_units( $val )
  returns: number

Parameters

  • @param: $val
  type: unit (pixel, em, rem)

    The unit you want striped to a number value.

px_to_em

Converts a pixel value to EMs. By default it'll use the base font size, but can be passed a custom base font size.

@function px_to_em( $px, $base )
  returns: unit (em)

Parameters

  • @param: $px
  type: unit (pixel, number)

    The pixel unit you want converted to EMs.

  • @param: $base
  type: unit (pixel, number)
  default: $base-font-size
    Reference: $base-font-size

    The base font-size that the returned EM value is calculated from.

Example Usage

SCSS 
h1 {
  font-size: px_to_em(28px);
}

.aside {
  font-size: px_to_em(12px);
  h2 {
    font-size: px_to_em(18px, 12px);
  }
}
CSS 
h1 {
  font-size: 2em;
}

.aside {
  font-size: 0.85714em;
}

.aside h2 {
  font-size: 1.5em;
}

em_to_px

Works the same as @function px_to_em except it converts an EM value to pixels. By default it'll use the base font size, but can be passed a custom base font size.

@function em_to_px( $em, $base )
  returns: unit (pixel)

Parameters

  • @param: $em
  type: unit (em, number)

    The EM unit you want converted to pixels.

  • @param: $base
  type: unit (pixel, number)
  default: $base-font-size
    Reference: $base-font-size

    The base font-size that the returned pixel value is calculated from.
