Horizontal Rules

Horizontal rules are elements that are written like this: <hr> and can be used in markup to breakup sections. Optionally, you can provide classes on the element to change the styles.

Default: <hr>

Dotted: <hr class="dotted">

Dashed: <hr class="dashed">

Double: <hr class="double">

Definition

Mobile First
The idea that web sites should be designed for mobile devices first, including only those tasks/items that website visitors use most. Then as screen real estate increases, add in tasks/features as needed based on user priority.
Progressive Enhancement
a strategy for web design that emphasizes accessibility, semantic HTML markup, and external stylesheet and scripting technologies.
Graceful Degradation
The property that enables a system (often computer-based) to continue operating properly in the event of the failure of (or one or more faults within) some of its components.
